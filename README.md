# NTHU Labook: frontend

## usage
### 0. Install the packages
```
npm install
```
### 1. Copy DB
Copy `db/graduate.db` and `db/list.yml` of scraper and paste to the `./db/` folder.

### 2. Run in localhost:5000
```
node app.js
```
